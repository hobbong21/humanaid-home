# CHON - Relationship is Identity

![CHON Logo](images/chon-logo.png)

## 📱 프로젝트 소개

CHON은 인간 관계를 핵심으로 하는 차세대 탈중앙화 신원 인증 플랫폼(DID, Decentralized Identity)입니다. 메타버스 시대를 위한 새로운 신원 인프라를 구축하여 **"관계가 곧 신원이다"** 라는 철학을 실현합니다.

## 🏢 HumanAID 소개

**HumanAID** (Human + AI + Digital)는 AI와 디지털 신원 기술을 결합하여 헬스케어, 스마트팜, DID 분야에 특화된 End-to-End AI 플랫폼을 제공합니다.

### 핵심 플랫폼: Make.IT_AI-Assistant

데이터 수집부터 모델 운영(MLOps)까지 전 과정을 지원하는 AI 분석 플랫폼으로, 헬스케어 AI, DID, 스마트팜 AI 솔루션의 기반이 됩니다.

#### 4단계 프로세스

1. **DATA Collection (데이터 수집)**
   - IoT Sensors, Database (SQL/NoSQL)
   - EMR / PACS (의료 데이터)
   - File Systems

2. **AI Modeling (AI 모델링)** ⭐ 핵심 단계
   - Data Preprocessing (데이터 전처리)
   - AutoML / LLM Training
   - Model Evaluation
   - 프롬프트·워크플로 빌더

3. **MLOps (운영 관리)**
   - Deploy → Monitor → Retrain 순환 구조
   - 드리프트 감지 및 재학습 자동화
   - 감사로그 및 모니터링

4. **Application (대시보드 & API)**
   - 실시간 대시보드
   - 경보 시스템
   - 경량 앱 배포

#### 차별화 포인트

- **도메인 특화 패키지**: 헬스케어, DID, 스마트팜 템플릿 사전 탑재
- **유연한 배포 환경**: 클라우드·온프레미스·엣지 지원
- **엔터프라이즈 보안**: 데이터 마스킹, 암호화, 접근제어

#### 핵심 성과 지표

- **분석 속도**: 3x Faster
- **운영 비용**: 40% Saving
- **데이터 사일로**: Zero

## 🏥 사업 분야 (1) - 헬스케어 AI & DID

### 헬스케어 AI 솔루션

#### 주요 기능
- **의료영상 분석**: CT/MRI/X-ray 병변 탐지·분류 보조
- **임상 의사결정 지원**: 진단·처방 보조, 예후·재원일 예측
- **운영 최적화**: 수요 예측, 스케줄링, 행정 자동화

#### 시장 규모
- **2026년**: $4.5B → **2035년**: $62.1B (약 14배 성장)
- **ROI**: 헬스케어 임원 90%가 AI 투자에서 긍정적 ROI 보고

#### 성공 사례
- **안중백병원**: 대웅제약과 AI 기반 병동 모니터링 시스템 도입
  - 실시간 환자 모니터링으로 선제적 대응
- **서울대병원**: SNUH.AI 의료 AI Agent 플랫폼
  - 반복 업무 자동화, 의료진 부담 경감

### DID (탈중앙화 신원증명)

#### 핵심 원칙
- **개인정보 주권**: 사용자가 자신의 데이터를 완전히 통제
- **위변조 방지**: 블록체인 기반 신원 정보 보호
- **상호운용성**: 다양한 플랫폼 간 신원 정보 호환

#### 주요 기능
- 자격 발급(Issuer) → 보관(Holder) → 검증(Verifier) 워크플로
- 선택적 정보 제공

#### 활용 분야
- 병원 체크인/원격진료 본인확인
- 임상시험 참여자 인증
- 스마트시티 출입/서비스 연동

#### 시장 규모
- **2026년**: $7.4B → **2031년**: $58.7B
- **CAGR**: 88% (초고속 성장)

#### 파일럿 프로젝트
- **라온시큐어**: 미국 CVS헬스·일본 히타치 파일럿
- **DTX 디지털 신원 인증 플랫폼** (ADI 스펙 V1.0)
- **KISA K-BTF 실증사업**: 공공기관 블록체인 공동 인프라
- **MyID 2.0** 모바일 인증서 발급/검증
