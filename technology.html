SERVICE OVERVIEW

서비스 개요

Decentralized Identity

관계 기반 신뢰 네트워크로
진정한 디지털 신원을 구축합니다

CHON DID는 Web of Trust 개념을 기반으로 한 혁신적인 탈중앙화 신원 인증 플랫폼입니다. 지인과 가족의 증명을 통해 Bottom-up 방식으로 신뢰를 구축하며, 영지식 증명(ZKP)으로 개인정보를 완벽하게 보호합니다.

Web of Trust

지인과 가족의 증명을 통한 관계 기반 신뢰 네트워크

ZKP Privacy

영지식 증명으로 민감정보를 노출하지 않고 신원 검증

Bottom-up DID

중앙 기관 없이 개인 간 신뢰를 쌓아가는 분산 신원

Data Asset

연결된 관계 데이터를 가치 있는 자산으로 전환

KEY BENEFITS

핵심 이점

01

개인정보 주권 확보

중앙 서버에 의존하지 않고 개인이 직접 신원 정보를 관리하며, 언제든지 접근 권한을 통제할 수 있습니다.

02

강화된 보안

블록체인 기술과 암호화로 해킹이나 데이터 유출로부터 안전하게 보호됩니다.

03

간편한 인증

한 번의 신원 등록으로 여러 서비스에서 간편하게 인증할 수 있습니다.

04

비용 절감

반복적인 신원 확인 절차를 줄여 시간과 비용을 절감합니다.

TECHNOLOGY

신원 인증 기술

CHON DID의 핵심 기술

최첨단 블록체인 기술과 암호화 알고리즘을 기반으로 안전하고 신뢰할 수 있는 탈중앙화 신원 인증 시스템을 구축했습니다.

Blockchain Infrastructure

  • 분산 원장 기술 (DLT)
  • 스마트 컨트랙트
  • 합의 알고리즘
  • 노드 네트워크

Security & Cryptography

  • 공개키 기반 구조 (PKI)
  • 영지식 증명 (Zero-Knowledge Proof)
  • 다중 서명 (Multi-Signature)
  • 암호화 해시 함수

DID Standards

  • W3C DID 표준 준수
  • Verifiable Credentials (VC)
  • DID Document 관리
  • Universal Resolver

User Experience

  • 생체 인증 (Biometric)
  • QR 코드 인증
  • 모바일 지갑
  • 크로스 플랫폼 지원
HOW IT WORKS

작동 원리

1

DID 생성

사용자가 앱을 통해 고유한 탈중앙화 식별자(DID)를 생성합니다

2

신원 정보 등록

개인 신원 정보를 암호화하여 블록체인에 안전하게 등록합니다

3

자격증명 발급

신뢰기관으로부터 검증 가능한 자격증명(VC)을 발급받습니다

4

선택적 공유

필요한 정보만 선택하여 서비스 제공자와 안전하게 공유합니다

5

신원 검증

블록체인을 통해 신원 정보의 진위 여부를 즉시 검증합니다

USE CASES

활용 사례

금융 서비스

은행 계좌 개설, 대출 신청 시 간편하고 안전한 본인 인증

교육 기관

학위, 자격증 등 학력 증명서의 디지털 발급 및 검증

의료 서비스

의료 기록 접근 권한 관리 및 안전한 정보 공유

정부 서비스

전자정부 서비스 이용 시 통합 신원 인증

전자상거래

온라인 쇼핑 시 안전한 결제 및 본인 확인

기업 서비스

직원 신원 관리 및 접근 권한 통제
