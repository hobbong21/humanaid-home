MAKE.IT_AI-ASSISTANT

핵심 플랫폼 기술

End-to-End AI Analytics Platform

데이터 수집부터 모델 운영까지
전 과정을 지원하는 통합 AI 플랫폼

Make.IT_AI-Assistant는 4단계 프로세스(Data Collection → AI Modeling → MLOps → Application)로 구성된 End-to-End AI 분석 플랫폼입니다. 도메인 특화 패키지, 유연한 배포 환경, 엔터프라이즈 보안을 제공하여 분석 효율 향상, 운영 최적화, 데이터 연계 강화를 실현합니다.

Data Collection

IoT 센서, 데이터베이스, EMR/PACS, 파일 시스템 등 다양한 소스에서 데이터를 수집합니다.

AI Modeling

데이터 전처리, AutoML/LLM 학습, 모델 평가, 프롬프트·워크플로 빌더를 제공합니다.

MLOps

배포 → 모니터링 → 재학습 순환 구조로 드리프트 감지 및 자동 재학습을 지원합니다.

Application

실시간 대시보드, 경보 시스템, 경량 앱 배포로 AI 모델을 실제 업무에 적용합니다.

KEY DIFFERENTIATORS

차별화 요소

01

도메인 특화 AI 패키지

헬스케어, DID, 스마트팜 산업에 최적화된 AI 템플릿을 사전 탑재하여 빠른 도입과 적용이 가능합니다.

02

유연한 배포 환경

클라우드, 온프레미스, 엣지 환경을 모두 지원하여 고객의 인프라 요구사항에 유연하게 대응합니다.

03

엔터프라이즈 보안

데이터 마스킹, 암호화, 접근제어 등 엔터프라이즈급 보안 기능을 기본 제공합니다.

04

실증된 ROI

실제 현장에서 검증된 성과를 바탕으로 생산성과 운영 효율 향상에 기여합니다.

DID TECHNOLOGY

탈중앙화 신원인증 (DID)

빠르게 성장하는 DID 시장에 대응하는 플랫폼 기술을 제공합니다.

DID 플랫폼 핵심 원칙

개인정보 주권, 위변조 방지, 상호운용성을 기반으로 안전하고 편리한 신원 인증을 제공합니다. Issuer(자격 발급) → Holder(보관) → Verifier(검증) 워크플로로 선택적 정보 제공이 가능합니다.

활용 분야

  • 병원 체크인/원격진료 본인확인
  • 임상시험 참여자 인증
  • 스마트시티 출입/서비스 연동
  • 금융 서비스 KYC

파일럿 프로젝트

  • 라온시큐어: 미국 CVS헬스·일본 히타치 파일럿
  • DTX 디지털 신원 인증 플랫폼 (ADI 스펙 V1.0)
  • KISA K-BTF 실증사업: 공공기관 블록체인 공동 인프라
  • MyID 2.0 모바일 인증서 발급/검증

핵심 기술

  • 블록체인 기반 분산 원장
  • 공개키 기반 구조 (PKI)
  • 영지식 증명 (Zero-Knowledge Proof)
  • W3C DID 표준 준수

보안 & 프라이버시

  • 선택적 정보 제공
  • 위변조 방지
  • 개인정보 주권 보장
  • 상호운용성 지원
HOW IT WORKS

작동 원리

1

DID 생성

사용자가 앱을 통해 고유한 탈중앙화 식별자(DID)를 생성합니다

2

신원 정보 등록

개인 신원 정보를 암호화하여 블록체인에 안전하게 등록합니다

3

자격증명 발급

신뢰기관으로부터 검증 가능한 자격증명(VC)을 발급받습니다

4

선택적 공유

필요한 정보만 선택하여 서비스 제공자와 안전하게 공유합니다

5

신원 검증

블록체인을 통해 신원 정보의 진위 여부를 즉시 검증합니다

기업 서비스
